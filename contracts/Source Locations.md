# Source Locations

## Raw data

Raw data and original source material live in Google Drive.

Examples:

- PDFs
- screenshots
- exported articles
- email exports
- transcripts
- source documents
- images
- raw notes before processing

## Processing layer

Processed AI notes live in Notion, in the `AI Inbox` system.

## Protocol repo

Workflow rules and templates live in GitHub:

`bozothegrey/ai-inbox-protocol`

## Canonical knowledge layer

Reviewed permanent notes live in the Personal Wiki / Obsidian.

## Rule for Sergio and other agents

When creating notes, assume that raw source files should remain in Google Drive unless Stefano explicitly asks to import them into Notion or Obsidian.

When referencing a source, include:

- source name
- source type
- Google Drive location if known
- date accessed or processed
