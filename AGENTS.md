# AGENTS.md

This is the operating protocol for Stefano's AI Inbox workflow.

## Storage model

Raw/source material lives in Google Drive.

Notion is the AI-operable inbox and working layer. It is used for processed notes, AI drafts, mobile captures, review queues, indexes, source summaries, working notes, and promotion candidates.

The main Personal Wiki / Obsidian remains the canonical destination for reviewed permanent notes.

GitHub stores this protocol, templates, workflow rules, and agent instructions.

## Folder / system rules

- Google Drive = raw/source archive.
- Notion = AI Inbox and working layer.
- Obsidian / Personal Wiki = curated permanent knowledge base.
- GitHub = protocol and templates.

## Note format

All AI-generated notes should have:

- Clear title
- Type
- Status
- Review state
- Domain
- Source, if known
- Raw Location, if known
- Target, if known
- Body sections: Summary, Main Note, Links / Suggested Links, Source / Raw Location, Review / Next Actions

## Editing rules

Do not import raw files into Notion or Obsidian by default. Reference Google Drive raw locations instead.

Do not treat Notion as the final Personal Wiki. It is a cockpit and processing layer.

Only cleaned, reviewed material should become a promotion candidate for the Personal Wiki.

If working inside a repo or local export, read this file first before creating or modifying notes.
