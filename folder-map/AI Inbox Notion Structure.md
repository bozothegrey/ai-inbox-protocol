# AI Inbox Notion Structure

Expected Notion structure:

```text
Private
└── AI Inbox
    ├── AI Notes database
    ├── Dashboard
    ├── Protocol
    ├── Source Map
    └── Promotion Queue
```

## Database: AI Notes

Recommended properties:

| Property | Type | Purpose |
|---|---|---|
| `Name` | Title | Note title |
| `Type` | Select | Kind of note |
| `Status` | Status | Workflow stage |
| `Review` | Select | Review state |
| `Domain` | Multi-select | Knowledge/project domain |
| `Source` | URL or Text | Original source |
| `Raw Location` | URL or Text | Google Drive raw location |
| `Target` | Select or Text | Destination system |
| `Created` | Created time | Automatic timestamp |
| `Last Edited` | Last edited time | Automatic timestamp |

## Recommended views

- Inbox
- Mobile Captures
- Needs Review
- Ready for Promotion
- By Domain
- Source Summaries
- Promoted / Archived
